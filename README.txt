QuickFill — Profile Autofill
============================

A simple browser extension that saves multiple personal profiles and fills
form fields (job portals, sign-up pages, etc.) with one click. Works in
Chrome, Brave, Edge, and other Chromium browsers.


HOW TO INSTALL (one time, ~30 seconds)
--------------------------------------
1. Unzip this file. You'll get a folder called "quickfill-extension".
2. Open your browser and go to the extensions page:
      Chrome  ->  chrome://extensions
      Brave   ->  brave://extensions
      Edge    ->  edge://extensions
3. Turn ON "Developer mode" (toggle, usually top-right).
4. Click "Load unpacked".
5. Select the "quickfill-extension" folder (the one with manifest.json inside).
6. Done. Click the puzzle-piece icon in the toolbar and pin QuickFill so it's
   always visible.


HOW TO USE
----------
1. Click the QuickFill icon.
2. Go to the "Profiles" tab -> "Add profile" -> fill in your details -> "Done".
   Add a second profile for a friend the same way; each person gets their own.
3. On the "Fill" tab, pick whose profile is active.
4. Open any form (e.g. a job application) and click "Fill this page".
   Or just press Alt+Shift+F on the page.

Tip: "Only fill fields that are empty" leaves anything you've already typed
untouched.

BACKUP & TRANSFER
-----------------
On the "Profiles" tab, use "Export" to save all your profiles to a JSON file,
and "Import" to load them back (e.g. on another computer or after reinstalling).
Imported profiles are added alongside any existing ones.


GOOD TO KNOW
------------
- All data stays on your computer (in the browser's local storage). Nothing is
  uploaded anywhere. There is no account and no server.
- Multiple profiles live side by side in the same browser — switch anytime.
- Passwords, search boxes, and checkboxes are never touched.
- It matches common fields (name, email, phone, address, LinkedIn, etc.). Very
  custom or unusually-labelled fields may not be recognised — just fill those
  by hand.
- To update your info later, reopen the popup and edit the profile.


SHARING
-------
You can send this zip to a friend; they install it the same way and create
their own profile. Profiles are not shared between computers.
